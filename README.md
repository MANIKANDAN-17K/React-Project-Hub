# ⚛️ React Project Hub

Welcome to **React-Project-Hub** 👨‍💻  
This repository contains small React projects built while learning React from **basic to advanced** concepts.

The goal of this repo is to master modern frontend development using React and build real-world applications.

---

## 📚 What This Repository Covers

This repository includes projects based on:

- ✅ React Basics (Components, JSX)
- ✅ Props & State
- ✅ Event Handling
- ✅ React Hooks (useState, useEffect, useRef, useContext)
- ✅ Conditional Rendering
- ✅ Forms & Controlled Components
- ✅ React Router
- ✅ API Integration
- ✅ Context API
- ✅ Performance Optimization
- ✅ Reusable Components & Clean Architecture

## 🚀 How to Run the Projects

1. Clone the repository:
```
git clone https://github.com/your-username/React-Project-Hub.git
```

2. Go inside a project folder:
```
cd project-folder-name
```

3. Install dependencies:
```
npm install
```

4. Start the development server:
```
npm start
```
